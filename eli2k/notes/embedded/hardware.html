Embedded Linux Refresher

A quick overview/refresher

Last updated: May 4th, 2023

Buy:

  • Raspberry Pi
  • Raspberry Pi Zero and Zero W
  • OrangePi
  • Other SBCs (mentioned sources: Hackerboards, Linux Gizmos, CNXSoftware)

Hack:

  • Consumer electronics running Linux (routers, IP cameras)
  • OpenWRT project
  • Reverse engineering an IP camera (example: Reolink IP camera)

Build:

  • Systems on a Module (SoM)
    • Onion Omega2S and Omega2
    • Other SoMs available
  • Bare processors
    • Nuvoton NUC980 series
    • Allwinner V3s
    • Allwinner F1C100s
    • Sunplus SP7021
    • Ingenic X1830
  • Recommended Tools:
    • FTDI breakout board (FT2232H) for UART communication, SPI flashing, JTAG control, etc.
    • SOP8 test clip for in-circuit programming of SPI flash
    • Miscellaneous supplies (header pins, breadboard wires, soldering iron, USB cables, SD cards)
